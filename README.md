Как проверять?
1) npm i
2) Прописать настройки MYSQL БД в .env
3) php artisan migrate
4) php artisan db:seed
5) php artisan serve

Запуск сервера
php artisan serve

Произведет настройки и наполнение БД:
php artisan migrate:refresh --seed
